Panduan Penggunaan

1. Fitur Data Splitter

  • Berfungsi memisahkan ratusan file laporan operasional berdasarkan kolom CODING.
  • Mapping Kategori: File otomatis terpisah menjadi Open.csv, Closed.csv, dan Claim.csv berdasarkan standarisasi kode perusahaan.
  • Blank Handling: Jika sel pada kolom CODING kosong, berisi spasi *invisible*, atau null/NaN, sistem otomatis membacanya sebagai "BLANK" dan memasukkannya ke file Open.
  • Auto-Chunking 500MB: Untuk mencegah Excel nge-lag saat dibuka, jika hasil pemisahan menembus ukuran 500MB, web akan memotongnya menjadi Open_1, Open_2, dst.

2. Fitur Data Merger

  • Berfungsi menumpuk/menggabungkan ratusan file *report* CSV/Excel menjadi satu file utuh.
  • Memiliki *engine* asinkron sehingga tidak menyebabkan browser *freeze*.
  • Jika hasil gabungan melebihi 500MB, file akan terpecah ke dalam beberapa *part*. Disarankan menggunakan format output CSV untuk stabilitas.
Privasi & Keamanan: Aplikasi Web ini 100% berjalan di atas RAM *Local-Client* komputer Anda. Tidak ada satupun file operasional rahasia perusahaan yang di-upload ke internet atau server luar.
